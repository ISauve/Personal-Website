Full Stack App Developer,
Waterloo Autonomous Car Project ("Autonomoose")

On my coop term in the summer of 2016, I developed a web app that students and staff at the University of Waterloo can use to order a ride from the autonomous shuttle driving around campus.

You can see more information on the project here. It was even featured on the Rick Mercer Report!

Features:

  • SimpleSAML authentication allowing only UWaterloo students/staff to access the service
  • Mobile- and desktop- compatible front end (built with Bootstrap)
  • Real-time display of the car location (animated with HTML Canvas)
  • Real-time display of approximate wait time to be picked up
  • Redis database storing user info such as number of rides taken, number of strikes, etc.
  • PHP back end which accepts ride requests; checks the user doesn't have too many strikes; adds requests to a real-time queue; and calculates ride information such as pickup time and position in queue
